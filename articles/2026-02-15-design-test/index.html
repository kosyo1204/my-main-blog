デザイン改善テスト記事

2026-02-15 デザイン

これは記事ページのデザイン改善をテストするための記事です。この最初の段落にはドロップキャップが適用され、最初の文字が大きく表示されるはずです。また、行長は65文字程度に制限され、読みやすい行間で表示されます。

見出し2のテスト

この見出しには左側に青いボーダーが表示されるはずです。見出しの前後には適切なマージンが設定されています。

見出し3のテスト

見出し3は青色で表示され、見出し2よりも小さく表示されます。

リストのテスト

順序なしリストのテスト：

  • 最初のリスト項目（カスタム記号が表示される）
  • 2番目のリスト項目
  • 3番目のリスト項目
    • ネストされたリスト項目

順序付きリストのテスト：

  1. 最初の項目
  2. 2番目の項目
  3. 3番目の項目

引用のテスト

これは引用ブロックです。背景色が薄い青色で、左側に青いボーダーが表示され、イタリック体で表示されます。引用ブロックは視覚的に際立つようにデザインされています。

コードブロックのテスト

インラインコード: const hello = "world";

コードブロック（コピーボタンが表示されるはず）:

function greet(name) {
  console.log(`Hello, ${name}!`);
  return `Welcome to ${name}'s blog`;
}

const result = greet('Visitor');
console.log(result);

別の例：

.article-content {
  max-width: 65ch;
  line-height: 1.8;
  margin: 0 auto;
}

テーブルのテスト

機能 状態 優先度
タイポグラフィ 実装済み High
コードコピー 実装済み High
ダークモード 実装済み Medium

画像のテスト

画像は角丸とシャドウ効果が適用されます（実際の画像がある場合）。

まとめ

この記事では、記事ページのデザイン改善の各要素をテストしました。すべての要素が適切にスタイル設定されていることを確認してください。
